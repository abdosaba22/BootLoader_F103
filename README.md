# BootLoader_F103
This Repo includes Bootloader App For STM32F103C8T6 and an App that Display an image on TFT  

# Mini Project ITI Arm Course: 
1- STM32F103C8T6.
2- FPEC driver.
3- STK driver.
4- UART driver.
5- DIO driver.
6- RCC driver.
7- TFT driver.
8- SPI driver.

# Used Tools:
1- IMT Sender Tool.
2- Stm32CubeIde.
